**Steam Inventory Parser**

Parser helps you get a list of all your items as JSON file. Works with open inventories only and any game.
Required:
1. Steam ID64
2. App Inventory ID

App IDs:
- Counter-Strike 2: 730
- Dota 2: 570
- TF2: 440
- Banana: 2923300
